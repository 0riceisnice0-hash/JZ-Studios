## Getting Started

Simply open `index.html` in your web browser to view the website. No build process or dependencies required!

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/0riceisnice0-hash/JZ-Studios.git
   ```

2. Navigate to the project directory:
   ```bash
   cd JZ-Studios
   ```

3. Open `index.html` in your browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # On Windows
   start index.html
   ```

## Customization

The website is built with flexibility in mind:

- **Colors**: Modify CSS variables in `styles.css` (`:root` section)
- **Content**: Edit text directly in `index.html`
- **Styling**: Update `styles.css` for design changes
- **Functionality**: Enhance `script.js` for additional features
